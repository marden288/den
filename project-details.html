Arduino Height Measuring System

This project is an Arduino-based height measuring system using an ultrasonic sensor, LED matrix, and PCB. It measures a person's height accurately and displays the result in real-time.

Features

  • Real-time height measurement
  • Ultrasonic sensor based detection
  • LED Matrix display
  • Compact PCB design

Technologies Used

  • Arduino
  • Ultrasonic Sensor
  • C / C++
  • Electronics
